# Government ITI College Website

A responsive and professional Government ITI College website built using **HTML, CSS, and JavaScript**.  
This project is designed to represent a real-world government institute website with a clean UI, proper structure, and student-friendly layout.

---

##  Project Overview

The Government ITI College Website provides complete information about the institute, available trades, admission process, and fee details.  
It follows a government-style design suitable for teachers, institutions, and official demonstrations.

---

##  Features

- Fully responsive design (Desktop & Mobile friendly)
- Government-style clean UI
- Admission form with structured inputs
- Passport photo upload with live preview
- Course / Trade information section
- Notice & announcement section
- Fee and payment information layout
- Contact details with helpline information
- Optimized layout for professional presentation
